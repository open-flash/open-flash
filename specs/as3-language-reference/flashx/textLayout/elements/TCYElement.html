The TCYElement (Tatechuuyoko - ta-tae-chu-yo-ko) class is a subclass of SubParagraphGroupElementBase that causes text to draw horizontally within a vertical line. Traditionally, it is used to make small blocks of non-Japanese text or numbers, such as dates, more readable. TCY can be applied to horizontal text, but has no effect on drawing style unless and until it is turned vertically. TCY blocks which contain no text will be removed from the text flow during the normalization process.
