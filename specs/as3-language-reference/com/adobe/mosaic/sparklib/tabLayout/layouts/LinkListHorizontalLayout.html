This layout is a modification of the ButtonBarHorizontalLayout. Normally, the buttons in a ButtonBar expand to fill the space so that if you have two buttons, they are both about 50% as wide as the ButtonBar. This layout removes that so each button is as wide as necessary but will still collapse in width if there isn't enough room for all of them.
