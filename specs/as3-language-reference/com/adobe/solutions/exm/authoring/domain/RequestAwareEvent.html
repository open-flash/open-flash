Product Version: Expression Manager Building Block 10
Runtime Versions: AIR (unsupported), Flash Player 10.2

Base class for events dispatched by various EXM domain authoring classes to indicate that an operation has returned or errored out. This type of event is relevant for those asynchronous operations which return a unique request ID to the caller, and itself contains the request ID corresponding to the original operation invocation, thus enabling the caller to make a correlation between the event and the invocation which triggered it.
