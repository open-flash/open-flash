The ParallelInstance class implements the instance class for the Parallel effect. Flex creates an instance of this class when it plays a Parallel effect; you do not create one yourself.

onEffectEnd

()method 
override protected function onEffectEnd(childEffect:IEffectInstance):void

Language Version: ActionScript 3.0
Product Version: Flex 3
Runtime Versions: Flash Player 9, AIR 1.1

Each time a child effect of SequenceInstance or ParallelInstance finishes, Flex calls the onEffectEnd() method. For SequenceInstance, it plays the next effect. In ParallelInstance, it keeps track of all the effects until all of them have finished playing. If you create a subclass of CompositeEffect, you must implement this method.

Parameters

childEffect:IEffectInstance — A child effect that has finished.
