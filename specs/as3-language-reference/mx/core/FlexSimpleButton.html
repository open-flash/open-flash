Language Version: ActionScript 3.0
Product Version: Flex 3
Runtime Versions: Flash Player 9, AIR 1.1

FlexSimpleButton is a subclass of the Player's SimpleButton class. It overrides the toString() method to return a string indicating the location of the object within the hierarchy of DisplayObjects in the application.
