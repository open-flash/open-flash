Classpublic class MessageNumbers
InheritanceMessageNumbers Object

Language Version: ActionScript 3.0
Product Version: Adobe Digital Enterprise Platform Document Services - Workspace 9
Runtime Versions: AIR (unsupported), Flash Player 9, Flash Player 10

The MessageNumbers class contains a list of error messages and their associated constant values. These messages are displayed as part of error messages in Workspace. Errors can occur in Workspace, Adobe Digital Enterprise Platform Document Services - Process Management 10.0 (found in the Adobe Digital Enterprise Platform Document Services - Administration Console), or the Document Server. More detailed information is found in the application server log files.
