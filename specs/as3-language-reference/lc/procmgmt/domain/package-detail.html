The lc.procmgmt.domain package contains classes that encapsulate process management concepts, also known as the business domain. The classes in this package are useful for applications built using Flex that require access to the Document server and Adobe Digital Enterprise Platform Document Services - Workspace data. Operations that are invoked may result in additional operations that occur on the Document server.
