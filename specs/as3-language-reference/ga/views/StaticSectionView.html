Implements IGuideView

Language Version: ActionScript 3.0
Product Version: Adobe Digital Enterprise Platform Document Services - Guides 9
Runtime Versions: AIR 1.0, Flash Player 10.2

The StaticSectionView class provides a dynamic list of the panels contained within a section. For example, using the following example Guide tree:
Guide  
 Section A 
  Panel A-1
  Panel A-2
 Section B  
  Panel B-1
If Section A is passed, this class provides the following panels:
  • Panel A-1
  • Panel A-2
When Section B is passed, this class provides the following panel:
  • Panel B-1
